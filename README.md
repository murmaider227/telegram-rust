## Installation

### git

* Install the rust toolchain in order to have cargo installed by following
  [this](https://www.rust-lang.org/tools/install) guide.
```shell
  git clone https://github.com/murmaider227/telegram-rust
  cd telegram-rust
  cargo build --release
  ```
